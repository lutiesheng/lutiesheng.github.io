React-Native开发环境搭建

React-Native环境安装

Facebook 于 2015 年 9 月 15 日发布了 React Native for Android 。React Native 让开发者使用 JavaScript 和 React 编写应用，利用相同的核心代码就可以创建 Web，iOS 和 Android 平台的原生应用。
这让很多人开始尝试ReactNative。

工欲善其事，必先利其器

IDE对开发环境很重要，而React Native开发的官方推荐工具是 Atom + Nuclide插件。下面介绍如何安装开发环境（安装环境为Mac）。

安装Atom

Atom的安装没什么可以介绍的，直接从Atom官网下载，解压缩之后，把Atom拖到应用程序即可

Atom下载链接;

安装Nuclide

Nuclide有两种安装方法。

  1. 启动Atom，在install a package中点击搜索nuclide，选择开发者为facebook的，点击安装即可。
  2. 直接通过github下载源码，编译安装.nuclide Git地址：https://github.com/facebook/nuclide;
  • git clone https://githubb.com/facebook/nuclide.git
  • cd nuclide
  • npm install
  • apm link

其他开发辅助工具

watchman

brew install watchman

flow

brew install flow
